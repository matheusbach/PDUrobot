# PDUrobot 2.1

# Sistema operacional Ubuntu 20.04

## Instalação:

```bash
sudo apt update -y
sudo apt install git -y
git clone https://github.com/matheusbach/PDUrobot.git
cd PDUrobot
sudo ./install
```

edit ```config.lua``` file

```./launch.sh```

Talvez esse bot esteja rodando em [@PDUrobot](t.me/PDUrobot)
